# Vector Addition Calculator

A web-based tool for visualizing and analyzing force vectors using matplotlib.

## Features
- Interactive vector visualization
- Real-time calculations  
- Step-by-step analytical solutions
- Mobile-friendly interface
- Matplotlib-based plots

## Usage
1. Enter force magnitudes and angles in sidebar
2. Click Calculate to visualize
3. Enable "Show Steps" for detailed solution

Built with Python, Streamlit, and Matplotlib.
