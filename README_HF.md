# Vector Addition Calculator

A web-based tool for visualizing and analyzing force vectors.

## Features
- Interactive vector visualization
- Real-time calculations
- Step-by-step analytical solutions
- Mobile-friendly interface
- Matplotlib-based plots

## Usage
1. Enter force magnitudes and angles
2. Click Calculate to visualize
3. Enable "Show analytical solution" for detailed steps

Built with Python, Gradio, and Matplotlib.
